# 📐 maquetacion77 - Frontend Component Ecosystem

Welcome! This is my space dedicated to web design and pure frontend development, focusing entirely on aesthetics, visual styles, and the creation of highly reusable components.

Here, you will find interactive elements and layout structures built exclusively with **HTML5, CSS3, and modern JavaScript**—ready to copy, paste, and adapt to any project without relying on heavy frameworks.

🚀 **View the live site:**[maquetacion77.github.io/maquetacion77](https://maquetacion77.github.io/maquetacion77/customLoaders.html)

## 🛠️ Project Status & Available Components

The site is currently under active development. The ecosystem will expand with new sections, but you can already explore and use the following premium-quality components:

*   **Hero Sections:**
    *   `hero01`: High-impact visual presentation structure.
    *   `hero02`: Modern layout variant optimized for conversion.
*   **Custom Loaders:**
    *   A collection of original, fluid loading animations optimized with CSS.

*(Coming soon: Full homepage with component search, dynamic buttons, interactive cards, navigation menus, and more).*

## 🎨 Code Features

*   **Pure & Lightweight:** No external dependencies (No React, No Tailwind, No Bootstrap). Maximum performance.
*   **Modular:** Designed around the "reusable components" principle. Clean code that is easy to isolate.
*   **Responsive Design:** Fully adapted for mobile devices, tablets, and desktop screens.
*   **Advanced CSS:** Uses variables (*Custom Properties*), Flexbox, CSS Grid, and optimized animations.

---

## 🚀 How to use these components

1. Explore the catalog on the live website.
2. Navigate within this repository to the folder for the component that interests you (e.g., `/hero01` or `/custom-loaders`). 3. Copy the HTML structure and the CSS stylesheet into your own project. And that's it!
